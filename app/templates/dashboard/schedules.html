Cron Expression Help

Cron expressions have 5 fields: minute hour day month weekday

Common Examples:
Expression Description
0 2 * * * Daily at 2:00 AM
0 */6 * * * Every 6 hours
0 9 * * 1-5 Weekdays at 9:00 AM
0 0 1 * * Monthly on the 1st at midnight
*/15 * * * * Every 15 minutes
0 12 * * 0 Weekly on Sunday at noon
Field Values:
  • Minute: 0-59
  • Hour: 0-23
  • Day: 1-31
  • Month: 1-12
  • Weekday: 0-6 (Sunday=0)
Special Characters:
  • * - Any value
  • / - Step values (e.g., */15 = every 15)
  • - - Range (e.g., 1-5)
  • , - Multiple values (e.g., 1,3,5)
